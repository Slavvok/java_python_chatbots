# ChatBots by Slavvok

Simple chatbot implemented in Python and Java. 

Fast lauch. 

Python:
  Cmd:
  ~ python ChatBotPy.py

Java:
In order to use in cmd, place 'includes' folder in 'java_chat/src'.
  1.Run from Eclipse. 
    File - Open Projects From File System.
    Run As java project. 
 2.Cmd:
  ~ javac ChatBot.java
  ~ java ChatBot
